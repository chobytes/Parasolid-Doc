Searching the Parasolid documentation

To search the Parasolid documentation:

  1. Type a word or phrase in the text box below
  2. Click the appropriate button (depending on whether you want to search for a single word, any word, or the exact phrase)
  3. Click Search.

A list of matches will be displayed in this window. Clicking on any of these matches will display the corresponding documentation in the main window.

Note: You can only search for whole words.

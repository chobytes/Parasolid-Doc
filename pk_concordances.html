About the concordances

The Parasolid documentation contains the following concordances, or word lists:

  • A concordance of PK interface names
  • A concordance of PK field names

Each word in the list is linked to a list of PK interface names that include the chosen word, each name in turn linked to the appropriate PK Reference entry.

How entries are compiled

As an example, consider the function PK_CIRCLE_ask, which receives a circle parameter and returns a circle_sf parameter.

  • The concordance of PK interface names contains two references to PK_CIRCLE_ask, based on the interface name, listed under circle and ask
  • The concordance of PK field names contains two references to PK_CIRCLE_ask, based on the parameter names, listed under circle and sf

Using the concordances

The concordances let you list all the PK interfaces which contain a given word, either in the PK interface name itself, or in the field names contained in that interface.

  • To see all the PK interfaces that contain the word clamp, look up clamp in the PK interface concordance
  • To see all the PK interfaces whose field names contain the word clamp, look up clamp in the PK field name concordance

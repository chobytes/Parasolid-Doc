Using the Parasolid Documentation Web

Welcome to the Parasolid Documentation Web. This site gives you access to the entire set of Parasolid manuals, in both HTML and PDF format. In addition, it provides a full text search facility, and an index of all PK interface names, as well as details on how to report problems to Parasolid Product Support.

Navigation bar

Along the top of this window is a navigation bar containing buttons that give you quick access to the most commonly used manuals. Alongside this is a list of all the manuals. To view the list of chapters for any manual, either click on the appropriate button, or choose the manual from the list and click Go >> .

To hide or view the menu (described below) click Display menu or Hide menu, as appropriate. Note that clicking this button returns you to the home page.

Menu

Buttons

Below the menu is a control panel with following options:

Search Search the Parasolid documentation. Clicking on this option opens a new window in which you can type a phrase. More instructions are available in the new window.
Help Displays this page.
Undock menu Displays the site menu in a separate browser window.
Dock menu Displays the site menu in the main browser window. Use this command to "dock" the menu back onto this window after using Undock menu..

This site is designed to be viewed with a Frames and JavaScript capable browser such as Microsoft Internet Explorer.

CRMINO   
Equivalent PK functions: PK_POINT_make_minimum_body
 
CRMINO ( minob, ifail )
=======================

  Create minimum object

Returns:
  KI_tag_body              *minob          --- new minimum object
  KI_cod_error             *ifail          --- failure indicator


Description:
  CRMINO creates a new body and attaches it to the world. The body is a minimum
  object, with just one shell, consisting of one vertex. The only
  geometry created will be a single point at (0, 0, 0), attached to the vertex.

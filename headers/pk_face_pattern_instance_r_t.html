struct PK_FACE_pattern_instance_r_s
    {
    PK_pattern_status_t        status;          --- state of this instance of
                                                --- the patterning operation
    int                        n_result_faces;  --- number of result faces
    PK_FACE_t                 *result_faces;    --- result faces for this
                                                --- instance of the pattern
    int                       *original_faces;  --- original pattern faces
                                                --- corresponding to result
                                                --- faces
    PK_TRANSF_t                transform;       --- transform which produced
                                                --- this instance
    int                        n_error_topols;  --- number of topological
                                                --- entities associated with
                                                --- pattern failure in this
                                                --- instance
    PK_TOPOL_t                *error_topols;    --- topological entities
                                                --- associated with pattern
                                                --- failure in this instance
    };
typedef struct PK_FACE_pattern_instance_r_s PK_FACE_pattern_instance_r_t;



This data structure contains the status and the patterning information
for a single instance of the pattern.

Used in:

PK_FACE_pattern_2_r_t


Description of fields:



'status'            This status field indicates the state of this
                    instance of the pattern. Permitted values are :

                     PK_pattern_status_ok_c
                     PK_pattern_status_fail_c
                     PK_pattern_status_outside_fa_c
                     PK_pattern_status_outside_reg_c
                     PK_pattern_status_tf_failed_c
                     PK_pattern_status_face_clash_c



'n_result_faces'    The length of the 'result_faces' array.


'result_faces'      The faces of this instance of the pattern which originated
                    from the input pattern faces. In general there will not be
                    a one-to-one correspondence between the 'result_faces'
                    array and the 'pattern_faces' array passed to
                    PK_FACE_pattern_2 as faces in the new instances may have
                    resulted from splitting or merging the images of original
                    pattern faces. For more detailed information about the
                    result faces, see the documentation accompanying
                    PK_FACE_pattern_2.


'original_faces'    The original pattern faces that produced the faces of the
                    new pattern instance in the 'result_faces' array. There is
                    a one-to-one correspondence between the 'original_faces'
                    array and the 'result_faces' array.


'transform'         The transform passed to PK_FACE_pattern_2 that produced
                    this instance of the pattern.


'n_error_topols'    The number of topological entities in the 'error_topols'
                    array.


'error_topols'      If the 'status' field of this return structure indicates a
                    failure, this array may contain topological entities which
                    are associated with the failure.

PK_CURVE_find_vectors_r_t    

struct PK_CURVE_find_vectors_r_s
    {
    int                          n_entries;
    int                         *indices;
    PK_CURVE_find_vectors_t     *statuses;
    PK_VECTOR_t                 *vectors;
    double                      *parms;
    };
typedef struct PK_CURVE_find_vectors_r_s PK_CURVE_find_vectors_r_t;



This return structure contains the return from PK_CURVE_find_vectors

Used in:

PK_CURVE_find_vectors
PK_CURVE_find_vectors_r_f



If PK_CURVE_find_vectors successfully determines a position vector
then:

    - The position vector on the curve and its corresponding parameter is
      returned in 'vectors' and 'parms' respectively. Only position vectors
      whose parameter lies within 't_int' will be returned.
    - The index of the value in 'measurements' that the position vector
      corresponds to is stored in 'indices'. Depending on 'measurement_method'
      a value in 'measurements' may correspond to several position vectors,
      in these cases all position vectors will be returned with the same
      index appearing multiple times in 'indices'.
    - The status field will hold PK_CURVE_find_vectors_ok_c to indicate
      that the position vector was successfully found.

If PK_CURVE_find_vectors found a position vector the error bound of which
exceeds the requested tolerance, but is less than ten times that tolerance
then:

    - The status field will hold PK_CURVE_find_vectors_tol_c to indicate
      that the solution is only approximate.
    - The position vector and its corresponding parameter are returned in
      'vectors' and 'params' respectively. Only position vectors whose
      parameter lies within 't_int' will be returned.
    - The index of the value in 'measurements' for which only an approximate
      solution could be found will be returned.

If PK_CURVE_find_vectors was unsuccessful in determining a position
vector then:

    - The index of the value in 'measurements' for which the computation
      failed will be returned.
    - The status field will indicate that an error did occur, see the
      documentation of PK_CURVE_find_vectors_t for more information.


The fields are as follows:

'n_entries'     Length of the arrays 'indices', 'statuses', 'vectors', 'parms'.

'indices'       Indices matching the returned values to the entries in
                'measurements' in the option structure.

'statuses'      Statuses corresponding to each of the position vectors in
                'vectors'.

'vectors'       Position vectors on the curve.

'parms'         Parameters corresponding to the position vectors in 'vectors'.
                The values in 'parms' will lie in the interval 't_int'.

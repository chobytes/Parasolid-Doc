PK_CURVE_degens_t    

struct PK_CURVE_degens_s
    {
    int               n_degen; --- number of degeneracies
    PK_CURVE_degen_t *degen;   --- details of degeneracies
    };

typedef struct PK_CURVE_degens_s PK_CURVE_degens_t;



A structure packaging details of degeneracies on a curve.

Used in:

PK_CURVE_degens_f
PK_CURVE_find_degens
PK_CURVE_fix_degens


The structure has the following fields:

'n_degen'    The number of degeneracies on the curve.

'degen'      An array of length 'n_degen', holding information on each
             degeneracy.

The function PK_CURVE_degens_f may be used to free associated memory.

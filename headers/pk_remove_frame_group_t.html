PK_remove_frame_group_t    

typedef int PK_remove_frame_group_t;

Used in:

PK_REGION_remove_lattice_o_t


This datatype specifies the group membership behaviour when
removing a lattice from a region.

PK_remove_frame_group_t can take the values:

    PK_remove_frame_group_remove_c   Remove the frame from all groups for
                                     which it is a member. This is the default
                                     behaviour.

    PK_remove_frame_group_replace_c  Replace the frame with the region in all
                                     all groups where the frame is a member
                                     and which may contain both frames and
                                     regions.

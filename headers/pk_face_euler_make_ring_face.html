PK_FACE_euler_make_ring_face    

PK_ERROR_code_t  PK_FACE_euler_make_ring_face
(
--- received arguments ---
PK_FACE_t        face,      --- Face to contain new face

--- returned arguments ---
PK_FACE_t *const new_face   --- New face created
)


This function adds a new ring loop to a face, creating a new face.


This function creates a ring edge which splits 'face' into two.  The new face
lies to the right of the new edge.

There will be no geometry attached to the new edge or face so the resulting
body will be invalid.

The inverse of this function is PK_EDGE_euler_delete_ring_face.

PK_detail_perforated_t    

typedef int PK_detail_perforated_t;

Used in:

PK_BODY_identify_details_o_t


This datatype determines whether a detail can be perforated, i.e. contain
inner boundaries of convex edges.

It has the values:

    PK_detail_perforated_no_c       Do not allow perforated details
    PK_detail_perforated_yes_c      Allow perforated details

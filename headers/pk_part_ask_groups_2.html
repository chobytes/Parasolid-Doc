PK_PART_ask_groups_2    

PK_ERROR_code_t               PK_PART_ask_groups_2
(
--- received arguments ---
PK_PART_t                     part,        --- a part
const PK_PART_ask_groups_o_t *options,     --- options

--- returned arguments ---
int                    *const n_groups,    --- number of groups (>=0)
PK_GROUP_t            **const groups       --- groups
)


This function returns the set of 'groups' contained within the given 'part'
satisfying conditions defined in the options structure.


Specific Errors:
    PK_ERROR_child_body             (MILD) 'part' is a child body

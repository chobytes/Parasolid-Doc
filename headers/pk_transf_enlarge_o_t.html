struct PK_TRANSF_enlarge_o_s
    {
    int                   o_t_version; --- version number of option structure
    PK_LOGICAL_t          modify;      --- whether to modify 'transf'
                                       --- (PK_LOGICAL_false)
    };

typedef struct PK_TRANSF_enlarge_o_s PK_TRANSF_enlarge_o_t;



This structure holds optional controls for enlarging a transform.

Used in:

PK_TRANSF_enlarge


The option structure has the following fields :

'modify'         If set to PK_LOGICAL_false, a new transform will be returned,
                 and the input transform will not be modified. If set to
                 PK_LOGICAL_true, the received 'transf' will be modified.
                 (PK_LOGICAL_false)

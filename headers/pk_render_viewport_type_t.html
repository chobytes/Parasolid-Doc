PK_render_viewport_type_t    

typedef int PK_render_viewport_type_t;

Used in:

PK_TOPOL_render_line_o_t


This typedef controls the type of viewports the application is passing in.
If it is set to 'PK_render_viewport_type_3D_c' then any non-aligned box can be
passed in to 'viewport_sf' or 'viewports'
If it is set to 'PK_render_viewport_type_2D_c' then any non-aligned boxes
passed to 'viewport_sf' or 'viewports' are interpreted as a 2D window using the
x and y axes, and the 3rd axis (which should be considered unbounded) should
be coincident with the view direction.

This field can take any one of the following values.
    'PK_render_viewport_type_3D_c' : 3D viewport (default)
    'PK_render_viewport_type_2D_c' : 2D viewport

PK_LATTICE_graph_t    

typedef int PK_LATTICE_graph_t;

Used in:

PK_LATTICE_graph_data_t


This datatype specifies the format for the block of data read in a
PK_LATTICE_graph_cb_f_t function call.

It can take the values:

    PK_LATTICE_graph_cyl_c          Data is represented in
                                    PK_LATTICE_graph_cyl_o_t format.

    PK_LATTICE_graph_cone_c         Data is represented in
                                    PK_LATTICE_graph_cone_o_t format.

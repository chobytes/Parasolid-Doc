PK_CURVE_ask_edges    

PK_ERROR_code_t   PK_CURVE_ask_edges
(
--- received arguments ---
PK_CURVE_t        curve,            --- a curve

--- returned arguments ---
int        *const n_edges,          --- number of edges (>= 0)
PK_EDGE_t **const edges             --- edges (optional)
)


This function returns the set of accurate edges to which the given curve is
attached.


The output array 'edges' is optional. If it is set to NULL, the function
will only compute the number of edges.

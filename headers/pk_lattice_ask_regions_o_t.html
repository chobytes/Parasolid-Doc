PK_LATTICE_ask_regions_o_t    

struct PK_LATTICE_ask_regions_o_s
    {
    int             o_t_version;    --- version number of options structure
    PK_LOGICAL_t    want_senses;    --- whether the senses of the lattice in
                                    --- the regions should be returned.
                                    --- (PK_LOGICAL_false)
    PK_LOGICAL_t    want_frames;    --- whether frames should be returned.
                                    --- (PK_LOGICAL_false)
    };

typedef struct PK_LATTICE_ask_regions_o_s PK_LATTICE_ask_regions_o_t;



Holds optional controls for querying which regions a lattice is embedded in.

Used in:

PK_LATTICE_ask_regions


The options provided are:

'want_senses'   Controls whether an array, 'senses', of the senses of the
                'lattice' in each region is returned. The array will be
                parallel to 'regions'.

'want_frames'   Controls whether an array, 'frames', of the frames embedding
                the 'lattice' in each region is returned. The array will be
                parallel to 'regions'.

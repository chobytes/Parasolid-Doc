PK_taper_step_face_t    

typedef int PK_taper_step_face_t;

Used in:

PK_FACE_change_taper_o_t
PK_FACE_taper_o_t


This datatype is used to specify if step faces need to be created between
taper faces. It has the values:

    PK_taper_step_face_yes_c   create a step face solution between taper faces
    PK_taper_step_face_no_c    don't create a step face solution
    PK_taper_preserve_smooth_c try to keep the smooth connectivity between
                               taper faces. Multiple taper angles cannot be
                               specified on smoothly connected faces

PK_outline_project_t    

typedef int PK_outline_project_t;

Used in:

PK_BODY_make_curves_outline_o_t
PK_BODY_make_persp_outline_o_t
PK_BODY_make_spun_outline_o_t


PK_outline_project_t can be set to one of the following values:

   PK_outline_project_no_c       Do not project the curves.

   PK_outline_project_plane_c    Project the curves onto a plane.

   PK_outline_project_sphere_c   Project the curves onto a sphere.

   PK_outline_project_cylinder_c Project the curves onto a cylinder.

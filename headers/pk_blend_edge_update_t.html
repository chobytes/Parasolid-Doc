PK_blend_edge_update_t    

typedef int PK_blend_edge_update_t;

Used in:

PK_BODY_fix_blends_o_t


This datatype en/dis-ables some changes to the blending algorithms which
might cause differences in model updates between versions.
It currently has the values:

    PK_blend_edge_update_0_c         for updating v9.1 to v12.1 models
    PK_blend_edge_update_1_c         for updating v13.0 models
    PK_blend_edge_update_2_c         for updating v13.2 to v15.0 models
    PK_blend_edge_update_3_c         for updating v15.1 to v16.0 models
    PK_blend_edge_update_4_c         for updating v16.1 models
    PK_blend_edge_update_5_c         for updating v17.0 models
    PK_blend_edge_update_6_c         for updating v17.1 models
    PK_blend_edge_update_7_c         for updating v18.0 models
    PK_blend_edge_update_8_c         for updating v18.1 models
    PK_blend_edge_update_9_c         for updating v19.0 models
    PK_blend_edge_update_10_c        for updating v19.1 models
    PK_blend_edge_update_11_c        for updating v20.0 models
    PK_blend_edge_update_12_c        for updating v21.0 models
    PK_blend_edge_update_13_c        for updating v22.0 to v22.1 models
    PK_blend_edge_update_14_c        for updating v23.0 models
    PK_blend_edge_update_15_c        for updating v23.1 models
    PK_blend_edge_update_16_c        for updating v24.0 models
    PK_blend_edge_update_17_c        for updating v24.1 models
    PK_blend_edge_update_18_c        for updating v25.0 models
    PK_blend_edge_update_19_c        for updating v25.1 models
    PK_blend_edge_update_20_c        for updating v26.0 models
    PK_blend_edge_update_v261_c      for updating v26.1 models
    PK_blend_edge_update_v270_c      for updating v27.0 models
    PK_blend_edge_update_v271_c      for updating v27.1 models
    PK_blend_edge_update_v280_c      for updating v28.0 models
    PK_blend_edge_update_default_c   use latest functionality

See Update control for more information.

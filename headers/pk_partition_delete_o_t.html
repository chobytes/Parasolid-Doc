PK_PARTITION_delete_o_t    

struct PK_PARTITION_delete_o_s
    {
    int          o_t_version;       --- version number
    PK_LOGICAL_t delete_non_empty;  --- forcibly delete non-empty partitions
                                    --- (PK_LOGICAL_false)
    };
typedef struct PK_PARTITION_delete_o_s PK_PARTITION_delete_o_t;



This structure contains optional arguments for PK_PARTITION_delete.

Used in:

PK_PARTITION_delete


Description of fields:


'delete_non_empty'  Delete partitions that are not empty and partitions that
                    might not be empty at a session mark. If this is set to
                    PK_LOGICAL_true then all model data in the partition, both
                    currently and at each mark, will be deleted. The default
                    value is PK_LOGICAL_false.

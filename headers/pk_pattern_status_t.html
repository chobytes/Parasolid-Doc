PK_pattern_status_t    

typedef int PK_pattern_status_t;

Used in:

PK_FACE_pattern_data_r_t
PK_FACE_pattern_instance_r_t


This datatype indicates the status for a particular instance of
the patterning operation. It has six values:

    PK_pattern_status_ok_c           no errors detected when pattern created
    PK_pattern_status_fail_c         errors detected when pattern created
    PK_pattern_status_outside_fa_c   transformed pattern outside base face
    PK_pattern_status_outside_reg_c  transformed pattern outside base region
    PK_pattern_status_tf_failed_c    transformation of pattern failed, possibly
                                     failed to regenerate SP-curve
    PK_pattern_status_face_clash_c   face-face clash detected
    PK_pattern_status_colliding_c    pattern instance is colliding with
                                     another instance or pattern feature

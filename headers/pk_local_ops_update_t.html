This datatype en/dis-ables some changes to local ops algorithms which
might cause differences in model updates between versions.  It currently has
the values:
    PK_local_ops_update_0_c         for updating v14.0 and earlier models
    PK_local_ops_update_1_c         for updating v14.1 - v16.1 models
    PK_local_ops_update_2_c         for updating v17.0 - v18.0 models
    PK_local_ops_update_3_c         for updating v18.1 - v21.0 models
    PK_local_ops_update_4_c         for updating v22.0 models
    PK_local_ops_update_5_c         for updating v22.1 models
    PK_local_ops_update_6_c         for updating v23.0 models
    PK_local_ops_update_7_c         for updating v23.1 models
    PK_local_ops_update_8_c         for updating v24.0 models
    PK_local_ops_update_9_c         for updating v24.1 models
    PK_local_ops_update_10_c        for updating v25.0 models
    PK_local_ops_update_11_c        for updating v25.1 models
    PK_local_ops_update_12_c        for updating v26.0 models
    PK_local_ops_update_v261_c      for updating v26.1 models
    PK_local_ops_update_v270_c      for updating v27.0 models
    PK_local_ops_update_v271_c      for updating v27.1 models
    PK_local_ops_update_v280_c      for updating v28.0 models
    PK_local_ops_update_default_c   use latest functionality

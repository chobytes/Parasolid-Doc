PK_APPITEM_create    

PK_ERROR_code_t     PK_APPITEM_create
(
--- received arguments ---
int                 n_pointers,   --- number of application pointers
const PK_POINTER_t  pointers[],

--- returned arguments ---
PK_APPITEM_t        appitems[]
)


This function creates 'appitems' to refer to the given 'pointers'.


For each given pointer in 'pointers', an APPITEM is created and returned in
the corresponding position in 'appitems'.

PK_BSURF_find_g1_discontinuity    

PK_ERROR_code_t PK_BSURF_find_g1_discontinuity
(
--- received arguments ---
PK_BSURF_t      bsurf,       --- bsurf

--- returned arguments ---
int      *const n_u_discs,   --- number of U discontinuities
double  **const u_discs,     --- surf U parameter values at
                             --- discontinuities
int      *const n_v_discs,   --- number of V discontinuities
double  **const v_discs      --- surf V parameter values at
                             --- discontinuities
)


This function finds G1 discontinuities on a B-surface.


This function will return the discontinuities of a B-surface.

The parameters will be returned in increasing parameter order in 'u_discs'
and 'v_discs'.

If there are no discontinuities on the geometry then 'n_u_discs' and/or
'n_v_discs' will be returned as zero.

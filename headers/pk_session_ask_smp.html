PK_SESSION_ask_smp    

PK_ERROR_code_t             PK_SESSION_ask_smp
(
--- returned arguments ---
PK_SESSION_smp_r_t   *const results      --- SMP return structure
)


This function retrieves the current session Symmetric Multi-Processing (SMP)
parameters.


Specific Errors:


The current SMP parameters are returned in the results structure. Refer
to the 'PK_SESSION_smp_r_t' documentation for information about the data
returned in this structure.

See also PK_SESSION_ask_smp_stacksize.

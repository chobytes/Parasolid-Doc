PK_SESSION_ask_latest_behaviour    

PK_ERROR_code_t               PK_SESSION_ask_latest_behaviour
(
--- returned arguments ---
PK_SESSION_behaviour_t *const latest_behaviour  --- latest behaviour
)


This function returns the latest allowed behaviour of the modeller regardless
of the current behaviour setting.


The 'latest_behaviour' will contain a 'behaviour_type' of
PK_SESSION_behave_as_latest_c and 'behaviour_value' of the latest integer
behaviour value.

See here for more information.

PK_sweep_guide_clamp_type_t    

typedef int PK_sweep_guide_clamp_type_t;

Used in:

PK_sweep_guide_control_t


This datatype is used to control the method used for clamping a guide wire in
sweeping.

It has the values:

    PK_sweep_guide_clamp_none_c     No clamp is applied.
    PK_sweep_guide_clamp_dirn_c     The swept cross-section will be transformed
                                    so that it preserves its angular
                                    relationship to the supplied clamp
                                    directions at each guide.
    PK_sweep_guide_clamp_fixed_c    The profile orientation will be locally
                                    fixed along the guide. In the case where
                                    the guide intersects the profile, this will
                                    have the effect that the swept surface will
                                    remain tangent with the profile at all
                                    points along the sweep. This option may not
                                    be used if 'guide_method' is set to
                                    PK_sweep_guide_curve_c, more than one
                                    profile is supplied, or the profile is a
                                    sheet body.

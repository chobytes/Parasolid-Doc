PK_CLASS_ask_superclass    

PK_ERROR_code_t   PK_CLASS_ask_superclass
(
--- received arguments ---
PK_CLASS_t        class,            --- a class

--- returned arguments ---
PK_CLASS_t *const superclass        --- immediate superclass of class
)


This function return the class token of the given class's immediate superclass.


PK_CLASS_class has no superclass; PK_CLASS_null will be returned in this case.

PK_TRANSF_transform    

PK_ERROR_code_t    PK_TRANSF_transform
(
--- received arguments ---
PK_TRANSF_t        transf_1,    --- first transformation
PK_TRANSF_t        transf_2,    --- second transformation

--- returned arguments ---
PK_TRANSF_t *const transf_out
)


This function evaluates the product of two transformations.

NOTE: This function is deprecated and has been superseded by
PK_TRANSF_transform_2.


'transf_out' is the result of multiplying 'transf_1' by 'transf_2'.
If used to transform an entity, 'transf_out' has the same effect as firstly
transforming the entity by 'transf_1' and then secondly by 'transf_2'.

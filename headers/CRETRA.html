CRETRA   
Equivalent PK functions: PK_TRANSF_create_translation
 
CRETRA ( direct, dist, transf, ifail )
======================================

  Create a translation transformation.

Receives:
  KI_vec_direction          direct         --- direction
  KI_dbl                   *dist           --- distance

Returns:
  KI_tag_transform         *transf         --- translation transformation
  KI_cod_error             *ifail          --- failure code


Description:
  Calculates a transformation matrix that, when applied to an entity,
  causes it to be translated in the given direction vector by the given
  distance.

PK_PARTITION_has_lattices_r_t    

struct PK_PARTITION_has_lattices_r_s
    {
    int           r_t_version;  --- version number of returns structure
    PK_LOGICAL_t  has_lattices; --- whether the partition contains lattices
    int           n_parts;      --- number of parts with lattice geometry
    PK_PART_t    *parts;        --- parts with lattice geometry
    int           n_lattices;   --- number of orphan lattices
    PK_LATTICE_t *lattices;     --- orphan lattices
    };

typedef struct PK_PARTITION_has_lattices_r_s PK_PARTITION_has_lattices_r_t;



A structure packaging the results of PK_PARTITION_has_lattices.

Used in:

PK_PARTITION_has_lattices
PK_PARTITION_has_lattices_r_f


Description of fields:

'has_lattices'   Whether the partition contains any orphan lattices, or parts
                 with embedded or construction lattices.

'n_parts'        The number of parts with lattice geometry in the 'partition'.
                 This will be set to 0 if 'want_parts' was set to
                 PK_LOGICAL_false.

'parts'          The parts in the 'partition' with lattice geometry.  This will
                 be set to NULL if 'want_parts' was set to PK_LOGICAL_false.

'n_lattices'     The number of orphan lattices in the 'partition'.  This will
                 be set to 0 if 'want_orphan' was set to PK_LOGICAL_false.

'lattices'       The orphan lattices in the 'partition'.  This will be set to
                 NULL if 'want_orphan' was set to PK_LOGICAL_false.

The function PK_PARTITION_has_lattices_r_f may be used to free associated
memory.

PK_APPITEM_t    



This type represents an APPITEM.  An APPITEM is an identifier, unique within a
session, which an application may request Parasolid to associate with some
application data.

Used in:

PK_APPITEM_ask
PK_APPITEM_create
PK_APPITEM_delete
PK_APPITEM_reset_pointers
PK_PARTITION_ask_appitems


When an APPITEM is created (PK_APPITEM_create) it is stored in the current
partition.  All APPITEMs in a partition may be obtained by
PK_PARTITION_ask_appitems.

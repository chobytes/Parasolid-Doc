PK_imprint_tracking_t    

typedef int PK_imprint_tracking_t;

Used in:

PK_BODY_imprint_cus_normal_o_t
PK_FACE_imprint_cus_normal_o_t


This datatype specifies the format of the tracking records that are returned
when imprinting curves

It has two values
    PK_imprint_tracking_basic_c    Default tracking structures used whereby
                                   returned information is as pre-V17.0.
    PK_imprint_tracking_curves_c   Detailed tracking records are returned
                                   recording the link between each imprinted
                                   edge and both the curve from which it
                                   originated and the topology upon which
                                   it is imprinted.

PK_MESH_cb_status_t    

typedef int PK_MESH_cb_status_t;


This datatype is the returned status of a callback function
used in a PK_MESH function.

Used in:

PK_MESH_facet_cb_f_t
PK_MFACET_cb_f_t
PK_MTOPOL_map_cb_f_t
PK_MTOPOL_select_cb_f_t
PK_MVERTEX_cb_f_t


It has the values:

    PK_MESH_cb_status_continue_c       operation may continue
    PK_MESH_cb_status_stop_c           operation is to stop
    PK_MESH_cb_status_memory_full_c    operation aborted because memory full
    PK_MESH_cb_status_fail_c           callback failed

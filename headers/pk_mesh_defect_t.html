PK_MESH_defect_t    

typedef int PK_MESH_defect_t;

Used in:

PK_MESH_defect_details_t


This datatype indicates the type of a mesh defect.
It has the values:

    PK_MESH_defect_corrupt_c            Mesh data is corrupt
    PK_MESH_defect_non_manifold_c       Mesh is non-manifold
    PK_MESH_defect_degen_mfacet_c       Mesh has a degenerate mfacet
    PK_MESH_defect_flat_mfacet_c        Mesh has a flat mfacet
    PK_MESH_defect_mvertex_normal_c     Mesh has an mvertex with a bad normal
    PK_MESH_defect_self_int_c           Mesh is self intersecting
    PK_MESH_defect_slit_c               Mesh has a slit
    PK_MESH_defect_foldover_c           Mesh has a foldover
    PK_MESH_defect_disjoint_c           Mesh is disjoint

PK_TOPOL_imprint_frames_o_t    

struct PK_TOPOL_imprint_frames_o_s
    {
    int             o_t_version;        --- version number of options structure
    double          distance_tolerance; --- tolerance for creating new facet
                                        --- faces (0.0)
    PK_FRAME_grow_region_t
                    grow_void_regions;  --- whether to extend or merge void
                                        --- material regions
                                        --- (PK_FRAME_grow_region_always_c)
    PK_FRAME_grow_region_t
                    grow_solid_regions; --- whether to extend or merge solid
                                        --- material regions
                                        --- (PK_FRAME_grow_region_always_c)
    };

typedef struct PK_TOPOL_imprint_frames_o_s PK_TOPOL_imprint_frames_o_t;



This structure contains optional controls for PK_TOPOL_imprint_frames.

Used in:

PK_TOPOL_imprint_frames


Description of fields:


'distance_tolerance'  This value controls the accuracy of conversion of a
                      frame's lattice to new facet faces.  When this option is
                      set to its default value Parasolid will find an
                      appropriate tolerance.


'grow_void_regions'   If this option is set to PK_FRAME_grow_region_always_c:
                          o Void material regions are extended into adjacent
                            imprinted frames whose senses are PK_LOGICAL_false.
                            If more than one region is extended into the same
                            frame, these regions are merged.
                          o Void material regions are merged with adjacent
                            regions whose material has changed from partial
                            void to void as a result of imprinting the frames.
                          o Adjacent regions are merged if their material has
                            changed from partial void to void as a result of
                            imprinting the frames.  Faces with the same region
                            on both sides undergoing this material change are
                            deleted.
                          o A single new void material region is created for
                            adjacent imprinted frame components whose senses
                            are PK_LOGICAL_false.

                      If this option is set to PK_FRAME_grow_region_no_c, no
                      extension or merging of void material regions occurs, and
                      imprinted frames in adjacent regions produce multiple
                      regions.

                      For more information on region material types, see
                      PK_REGION_material_t.


'grow_solid_regions'  If this option is set to PK_FRAME_grow_region_always_c:
                          o Solid material regions are extended into adjacent
                            imprinted frames whose senses are PK_LOGICAL_true.
                            If more than one region is extended into the same
                            frame, these regions are merged.
                          o Solid material regions are merged with adjacent
                            regions whose material has changed from partial
                            solid to solid as a result of imprinting the
                            frames.
                          o Adjacent regions are merged if their material has
                            changed from partial solid to solid as a result of
                            imprinting the frames.  Faces with the same region
                            on both sides undergoing this material change are
                            deleted.
                          o A single new solid material region is created for
                            adjacent imprinted frame components whose senses
                            are PK_LOGICAL_true.

                      If this option is set to PK_FRAME_grow_region_no_c, no
                      extension or merging of solid material regions occurs,
                      and imprinted frames in adjacent regions produce multiple
                      regions.

                      For more information on region material types, see
                      PK_REGION_material_t.

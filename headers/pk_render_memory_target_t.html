PK_render_memory_target_t    

typedef int PK_render_memory_target_t;

Used in:

PK_TOPOL_render_line_o_t


The 'memory target' option can be set to one of the following values:

PK_render_memory_target_yes_c       Parasolid will attempt to restrict the
                                    memory usage of PK_TOPOL_render_line to
                                    the amount of memory given by
                                    "memory_target_value".  The target value
                                    is not guaranteed to be met. The amount of
                                    memory used by PK_TOPOL_render_line will
                                    be reduced at the expense of total
                                    processor time.

                                    The value given by 'memory_target' refers
                                    to the peak amount of extra memory
                                    temporarily allocated during
                                    PK_TOPOL_render_line.

PK_render_memory_target_no_c        Parasolid will maximise performance without
                                    attempting to restrict memory usage.

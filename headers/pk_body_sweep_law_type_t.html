PK_BODY_sweep_law_type_t    

typedef int PK_BODY_sweep_law_type_t;

Used in:

PK_BODY_sweep_law_t


The value of this token indicates the method by which a twist or scale law
is being specified through the PK

    PK_BODY_sweep_law_none_c        The law structure is empty/unused.

    PK_BODY_sweep_law_discrete_c    The law is specified by giving a discrete
                                    set of values to be interpolated at the
                                    vertices of the path.

    PK_BODY_sweep_law_curve_c       The law is a continuous scalar valued
                                    function. It is specified by a
                                    1-dimensional B-spline curve parameterised
                                    [0 1] that evaluates over this range
                                    to the values of the scalar law. If the
                                    path is composed of a single C1 smooth edge
                                    then the parameter values are mapped
                                    linearly onto the path's parameterisation.

    PK_BODY_sweep_law_curve_inv_c   The law is a continuous scalar valued
                                    function. It is specified by a
                                    1-dimensional B-spline curve that
                                    represents the inverse of the function,
                                    i.e. it is parameterised over the values
                                    of the law and evaluates to values in the
                                    range [0 1]. If the path is composed of a
                                    single C1 smooth edge then the evaluated
                                    values are mapped linearly onto the path's
                                    parameterisation.

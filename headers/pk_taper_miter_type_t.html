PK_taper_miter_type_t    

typedef int PK_taper_miter_type_t;

Used in:

PK_BODY_taper_o_t


The values indicate how a taper surface may be further modified.

    PK_taper_miter_on_ref_c  :  Taper surfaces are mitered when necessary.
                                The resulting mitered faces remain bounded
                                by the supplied reference edges.
    PK_taper_miter_to_face_c :  Taper surfaces are mitered when necessary,
                                to contact tangentially the faces of the
                                reference edges on the side further from
                                the parting sheet (It is an error to set this
                                value if 'non_miter_edges' are supplied).
    PK_taper_miter_at_open_c :  Both above types of mitering may be used.
                                Type miter_to_face is applied at any open
                                parts of the chain formed by the reference
                                edges, where the faces to be tangent to the
                                mitered faces are those meeting the parting
                                sheet on the same side as the reference edges.
                                Type miter_on_ref is applied to the remaining
                                reference edges.

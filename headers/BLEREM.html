BLEREM   
Equivalent PK functions: PK_EDGE_remove_blend
 
BLEREM ( edges, ifail )
=======================

  Remove blend attributes.

Receives:
  KI_tag_list_edge         *edges          --- list of edges to remove blends from

Returns:
  KI_cod_error             *ifail          --- failure code


Description:
  Any blend attributes are removed from the edges.

This structure contains optional controls for PK_BODY_make_patterned.


Specific Errors:
    PK_ERROR_bad_value              'repetitions_i', 'repetitions_j', or
                                    'repetitions_k' within 'bound' have a
                                    first value greater than the second, or
                                    specify a number of repetitions greater
                                    than a complete period if periodic in that
                                    direction. (MILD)
    PK_ERROR_bad_trim_curves        Bad trim curves specified as follows:
                                    - Trim curves have been provided that do
                                      not lie in a plane spanning the i and j
                                      directions of a rectilinear or original
                                      pattern. (MILD)
                                    - Trim curves have been provided whose
                                      polyline positions have a non-zero z
                                      component in an axial pattern. (MILD)
                                    - Trim curves have been provided whose
                                      segments' projections are ambiguous when
                                      applied to an axial type pattern. (MILD)
    PK_ERROR_bad_form               An 'axial' 'form' has been provided with
                                    an insufficient distance between the
                                    'origin' and 'axis', to prevent the pattern
                                    from intersecting with the 'axis', when a
                                    callback hasn't been provided. (MILD)

PK_BODY_make_patterned

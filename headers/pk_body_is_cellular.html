PK_BODY_is_cellular    

PK_ERROR_code_t                PK_BODY_is_cellular
(
--- received arguments ---
PK_BODY_t                      body,    --- body
const PK_BODY_is_cellular_o_t *options, --- options

--- returned arguments ---
PK_BODY_is_cellular_r_t *const results  --- results
)


This function returns whether the given body is cellular.


A body is cellular if:

    - it does not contain any acorn vertices, wireframe edges, laminar edges or
      sheet faces, and
    - the set of outside faces form a manifold solid when all other faces are
      hidden, and
    - the set of solid faces form a manifold solid when all other faces are
      hidden.

Sheet faces are those between regions containing no solid material, either as a
whole region or within a lattice.

Outside faces are those between regions containing solid material, either
within the whole region or only within a lattice, and those containing none;
setting the cellular guise using PK_SESSION_set_cellular_guise to
PK_cellular_guise_outside_c will hide all faces but these.

Solid faces are those between solid regions and void regions, irrespective of
any lattices embedded in the regions; setting the cellular guise using
PK_SESSION_set_cellular_guise to PK_cellular_guise_solid_c will hide all faces
but these.

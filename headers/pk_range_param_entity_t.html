Option to return a vector and parameters on the sub-entity if a sub-entity has
been identified. This will always be the case for bodies. Additionally, this
option controls whether a fin can be returned as the sub-entity.

    PK_range_param_entity_topol_c     Vector and parameters on the entity
                                      (except bodies) will be returned
                                      irrespective of whether a sub-entity has
                                      been identified. Fins are never returned
                                      as the sub-entity, instead the fin's edge
                                      is returned.
    PK_range_param_entity_sub_c       Vector and parameters on the sub-entity
                                      will be returned if a sub-entity has been
                                      identified. Fins are returned as the
                                      sub-entity if identified as such.

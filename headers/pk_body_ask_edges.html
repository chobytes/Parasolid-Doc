PK_BODY_ask_edges    

PK_ERROR_code_t   PK_BODY_ask_edges
(
--- received arguments ---
PK_BODY_t         body,             --- a body

--- returned arguments ---
int        *const n_edges,          --- number of edges (>= 0)
PK_EDGE_t **const edges             --- edges (optional)
)


This function returns the set of edges in the given body.


Depending upon the configuration of the 'body' ( see PK_BODY_ask_config ) this
function returns:
    PK_BODY_config_standard_c      all edges of the standard 'body'

    PK_BODY_config_compound_c      all edges of all children of the
                                   compound 'body'

    PK_BODY_config_child_c         all edges of the child 'body'

The output array 'edges' is optional. If it is set to NULL, the function
will only compute the number of edges in the body.

See Data structure enquiries for related functions.

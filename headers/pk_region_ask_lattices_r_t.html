struct PK_REGION_ask_lattices_r_s
    {
    int           r_t_version;  --- version number of return structure
    int           n_lattices;   --- number of lattices
    PK_LATTICE_t *lattices;     --- the lattices
    PK_LOGICAL_t *senses;       --- the senses
    PK_FRAME_t   *frames;       --- the frames
    };
typedef struct PK_REGION_ask_lattices_r_s PK_REGION_ask_lattices_r_t;


A structure packaging the results of PK_REGION_ask_lattices.

Used in:

PK_REGION_ask_lattices
PK_REGION_ask_lattices_r_f


The structure has the following fields:

'n_lattices'    The number of lattices embedded in 'region'.

'lattices'      The lattices embedded in 'region'.

'senses'        The senses of the 'lattices' in the 'region'. `senses[i]' is
                the sense of `lattices[i]' in the 'region'. PK_LOGICAL_true
                indicates that the interior of `lattices[i]' is considered
                solid in the 'region'. PK_LOGICAL_false indicates that the
                interior of `lattices[i]' is considered void in the 'region'.
                This will be set to NULL if 'want_senses' was set to
                PK_LOGICAL_false, otherwise the array will be parallel to
                'lattices'. The sense can be reversed by calling
                the function PK_FRAME_reverse.

'frames'        The frames embedding the 'lattices' in the 'region'.
                `frames[i]' embeds `lattices[i]' in the 'region'. This will be
                set to NULL if 'want_frames' was set to PK_LOGICAL_false,
                otherwise the array will be parallel to 'lattices'.

The function PK_REGION_ask_lattices_r_f may be used to free associated memory.

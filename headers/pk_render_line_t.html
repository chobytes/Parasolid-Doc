PK_render_line_t    

typedef int PK_render_line_t;


This typedef controls the type of line for which a line record is
generated in the line fit tolerances report.

This field can take any one of the following values.
    PK_render_line_edge_c       : edge line
    PK_render_line_silh_c       : silhouette line
    PK_render_line_hatch_c      : hatch line
    PK_render_line_inter_c      : interference line
    PK_render_line_clip_c       : clip line

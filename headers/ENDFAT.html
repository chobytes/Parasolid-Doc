Equivalent PK functions: PK_ATTRIB_ask_attdef
 
ENDFAT ( attrib, type, ifail )
==============================

  Enquire the attribute type definition of an attribute.

    Can be called from the GO.

Receives:
  KI_tag_attribute         *attrib         --- attribute

Returns:
  KI_tag_attrib_def        *type           --- attribute type
  KI_cod_error             *ifail          --- error code


Description:
   Returns the attribute type definition of the given attribute.

This structure contains option controls for PK_TOPOL_find_box_2

Used in:

PK_TOPOL_find_box_2


The options provided are:

'want_topol_boxes'      whether the application requires individual boxes
                        for each topology.

'parts_with_scales'     This array, of length 'n_parts_with_scales', contains
                        unique parts which are to be considered at the scale
                        factors provided in 'scale_factors'. If this array
                        is NULL, no scale factors are applied.

'scale_factors'         This array should match the length of
                        'parts_with_scales'. Each element of the array should
                        be a PK_scale_factor_t corresponding to each part
                        specified in 'parts_with_scales'.

'output_scale_factor'   This is a scale factor used to shrink the
                        resulting box(es), where the origin is used as
                        the centre of scaling. No scaling is applied when the
                        value is set to PK_scale_factor_1x_c.


SCALE FACTORS
It is possible to associate a scale factor with a part so that any topology
in 'topols' that is provided from such a part is treated at the scale factor
of that part. The centre of scaling is the origin.

In order to supply scale factors for parts it is necessary to fill in the
following fields: 'parts_with_scales' and 'scale_factors'. Additionally,
'n_parts_with_scales' must be non-zero.

Topologies other than assemblies are treated with the scale factor of their
owner body. The topology's transformation, if given in 'transfs', is applied
after the owner body has been scaled. When a scale factor is provided for an
assembly, it will be applied only to the translation component of the
transforms of the assembly's constituent instances.  The scale factor will not
apply to the parts contained within these instances.  Optionally scale factors
can be supplied within 'parts_with_scales' and 'scale_factors' for any part
referenced within the assembly structure. In which case, these parts will be
scaled by the provided scale factor. In this way, scale factors can be supplied
to nested assembly structures, scaling bodies or other assemblies contained
within instances inside a parent assembly.

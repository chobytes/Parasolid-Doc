PK_outline_update_t    

typedef int PK_outline_update_t;

Used in:

PK_BODY_make_curves_outline_o_t
PK_BODY_make_spun_outline_o_t


This datatype en/dis-ables some changes to the outline algorithms which
might cause differences in model updates between versions.
It currently has the values:

    PK_outline_update_0_c         for updating v22.1 and earlier models
    PK_outline_update_1_c         for updating v23.0 models
    PK_outline_update_2_c         for updating v23.1 models
    PK_outline_update_3_c         for updating v24.0 models
    PK_outline_update_4_c         for updating v24.1 models
    PK_outline_update_5_c         for updating v25.0 models
    PK_outline_update_6_c         for updating v25.1 models
    PK_outline_update_7_c         for updating v26.0 models
    PK_outline_update_v261_c      for updating v26.1 models
    PK_outline_update_v270_c      for updating v27.0 models
    PK_outline_update_v271_c      for updating v27.1 models
    PK_outline_update_v280_c      for updating v28.0 models
    PK_outline_update_default_c   use latest functionality

PK_render_report_line_t    

typedef int PK_render_report_line_t;

Used in:

PK_TOPOL_render_line_o_t


This typedef controls the type of line report made for failing fits.

This field can take any one of the following values.
    PK_render_report_line_no_c         report nothing (default)
    PK_render_report_line_fail_c       report all line fit failures
    PK_render_report_line_loose_c      report line fit successes using
                                       looser tolerances
    PK_render_report_line_all_c        report line failures and line
                                       fits using looser tolerances

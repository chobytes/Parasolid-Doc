PK_EDGE_reverse    

PK_ERROR_code_t PK_EDGE_reverse
(
--- received arguments ---
PK_EDGE_t       edge  --- edge to reverse
)


This function reverses the given edge and its associated geometry.

NOTE: This function is deprecated and has been superseded by PK_EDGE_reverse_2.



Specific Errors:
    PK_ERROR_reverse_edge_failed    operation failed


All fins of the edge have their fin senses (as given by PK_FIN_is_positive)
reversed.  The order of the edge's fins is then changed to ensure that the
first fin (as given by PK_EDGE_ask_first_fin) has positive sense.

Any curves attached to the edge's fins are left unchanged.

If the edge itself has a curve attached (either the edge is accurate or has
a nominal curve), then that curve is detached from the edge, a reversed curve
is made from it (as by PK_CURVE_make_curve_reversed), and the reversed curve
attached to the edge.  If the original curve is not shared by other edges or
owned by other geometry then it is deleted.

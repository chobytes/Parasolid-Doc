PK_BCURVE_extend    

PK_ERROR_code_t PK_BCURVE_extend
(
--- received arguments ---
PK_BCURVE_t                 bcurve,    --- bcurve to extend
const PK_BCURVE_extend_o_t *options,   --- options structure

--- returned arguments ---
PK_BCURVE_extend_r_t *const results    --- results
)


This function extends a bcurve.


Specific Errors:
    PK_ERROR_bad_value              Indicates that an invalid value has been
                                    supplied in the options structure (MILD)

    PK_ERROR_bad_combination        Indicates that an invalid combination
                                    of values has been supplied in the
                                    options structure (MILD)



This function extends a bcurve. Options are available to specify how far the
bcurve should be extended, and what method should be used to extend it.

The bcurve may be attached to topology or be orphan.  The original bcurve will
be extended, unless either the bcurve has geometric owners, or it is a closed
curve attached to topology and the 'extend_closed' option has been set to allow
the extension of closed bcurves.  In these two cases the bcurve will be
copied, and the copy will be extended.  The parameterisation of the bcurve is
maintained.

The default options will result in no change to the bcurve and return a status
of PK_BCURVE_extend_unextended_c. Details of the options structure can be
found in its documentation.

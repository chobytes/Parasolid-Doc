PK_embed_frame_group_t    

typedef int PK_embed_frame_group_t;

Used in:

PK_REGION_embed_lattices_o_t


This datatype specifies the group membership behaviour when
embedding a lattice into a region.

PK_embed_frame_group_t can take the values:

    PK_embed_frame_group_no_c        No processing is performed on groups,
                                     this is the default.

    PK_embed_frame_group_add_c       Add the frame to all groups where the
                                     region is a member and which may contain
                                     both frames and regions

    PK_embed_frame_group_replace_c   Replace the region with the frame in all
                                     groups where the region is a member and
                                     which may contain both frames and regions.

    PK_embed_frame_group_remove_c    Remove the region from all groups where it
                                     is a member. The frame is not added to any
                                     groups.

PK_TOPOL_find_frames_r_t    

struct PK_TOPOL_find_frames_r_s
    {
    int           r_t_version; --- version number of returns structure
    int           n_frames;    --- number of frames
    PK_FRAME_t   *frames;      --- the frames
    PK_GEOM_t    *geoms;       --- the geometries
    PK_ENTITY_t  *owners;      --- the frame owners
    PK_LOGICAL_t *senses;      --- the frame senses
    };

typedef struct PK_TOPOL_find_frames_r_s PK_TOPOL_find_frames_r_t;



A structure packaging the results of PK_TOPOL_find_frames.

Used in:

PK_TOPOL_find_frames
PK_TOPOL_find_frames_r_f


Description of fields:

'n_frames'   The number of frames associated with 'topol'

'frames'     The frames associated with 'topol'

'geoms'      The geometries of the 'frames' associated with the 'topol'.
             `geoms[i]' is the geometry of `frames[i]'.
             This will be set to NULL if 'want_geoms' was set to
             PK_LOGICAL_false, otherwise the array will be parallel to
             'frames'.

'owners'     The owners of the 'frames' associated with the 'topol'.
             `owners[i]' is the owner of `frames[i]'.
             This will be set to NULL if 'want_owners' was set to
             PK_LOGICAL_false, otherwise the array will be parallel to
             'frames'.

'senses'     The senses of the 'frames' associated with the 'topol'.
             `senses[i]' is the sense of `frames[i]'.
             This will be set to NULL if 'want_senses' was set to
             PK_LOGICAL_false, otherwise the array will be parallel to
             'frames'.

The function PK_TOPOL_find_frames_r_f may be used to free associated memory.

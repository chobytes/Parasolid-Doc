PK_INSTANCE_t    



This type represents an INSTANCE.  An instance positions and orientates a
single occurrence of a part within an assembly.

Used in:

PK_ASSEMBLY_ask_instances
PK_INSTANCE_ask
PK_INSTANCE_change_part
PK_INSTANCE_create
PK_INSTANCE_replace_transf
PK_INSTANCE_transform
PK_PART_ask_ref_instances


An instance is a type of topology (PK_CLASS_instance is a subclass of
PK_CLASS_topol).

See Instance for more information.

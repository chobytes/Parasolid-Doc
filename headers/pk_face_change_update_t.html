PK_FACE_change_update_t    

typedef PK_local_ops_update_t   PK_FACE_change_update_t;


The default value of this datatype allows applications to take advantage of
the latest enhancements to the face change algorithm. To maintain consistency
when rebuilding models created in a previous version of Parasolid, a
non-default value should be passed.

    PK_FACE_change_update_0_c          for updating v18.0 and earlier models
    PK_FACE_change_update_default_c    activate latest functionality

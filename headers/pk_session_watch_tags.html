PK_SESSION_watch_tags    

PK_ERROR_code_t   PK_SESSION_watch_tags
(
--- received arguments ---
int                                 n_tags,     --- number of tags
                                                --- (may be zero)
const int                           tags[],     --- tags to watch
const PK_SESSION_watch_create_cb_t  create_fn,  --- create entity callback
                                                --- (may be NULL)
const PK_SESSION_watch_destroy_cb_t destroy_fn  --- destroy entity callback
                                                --- (may be NULL)
)


This function registers an array of tagged entities which are to be watched
via two application callback functions.

NOTE: This function is deprecated and has been superseded by
PK_DEBUG_SESSION_watch_items.




The application may register an array of tagged entities to watch via a
create and a destroy callback function. The create callback function
will be invoked whenever one of the watched entities is created. The
destroy callback will be invoked whenever one of the watched entities
is deleted.

This function may be called before or after the session has been
started (ie. before or after a call to PK_SESSION_start). This
function must not be invoked before a frustrum has been registered
via PK_SESSION_register_frustrum.

The system to watch entities may be disabled by invoking the function and
specifying zero entities to watch.


NOTE: This function is only to be used for debugging purposes, it
      not to be used to track the creation and deletion of tagged
      entities during general application execution.

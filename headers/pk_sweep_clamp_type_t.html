PK_sweep_clamp_type_t    

typedef int PK_sweep_clamp_type_t;

Used in:

PK_sweep_deriv_conds_t


This datatype is used to control the method used for clamping a profile in
sweeping.

It has the values:

    PK_sweep_clamp_none_c             The tangents across the profile are not
                                      constrained.
    PK_sweep_clamp_const_profile_c    The tangents across the profile are
                                      constrained to be equal to those of the
                                      swept solution that would result if only
                                      that profile was supplied. This option
                                      may only be used if there is more than 1
                                      profile.
    PK_sweep_clamp_face_c             The tangents across the profile are
                                      constrained so that they meet the
                                      supplied face smoothly.

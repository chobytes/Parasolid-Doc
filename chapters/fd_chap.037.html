Contents

Parasolid provides a wide range of generalised surfacing functionality that is used to create more sophisticated bodies from profiles and wires, as well as techniques that support advanced feature creation.This functionality is integrated at the topological level, which results in industry-leading functionality, reliability and performance. This volume explains how to create these bodies from a set of wires or by extruding a body in a linear direction. It also explains how these bodies are created by spinning profiles on an axis, sweeping profiles along a path, or lofting between a set of profiles. Information on how to create shadow curves and emboss features on a whole body or selected faces within a body is also provided.

This volume contains the following chapters:

    • Chapter 37, “Creating Profiles” describes how to create simple zero-, one- or two-dimensional bodies that are used to create new bodies with free-form geometry
    • Chapter 38, “Creating Extruded Bodies” describes how to create a body by linear extrusion of a profile body
    • Chapter 39, “Sweeping” describes how to create new bodies by sweeping entities along a path
    • Chapter 41, “Lofting” describes how to create a sheet or solid body by fitting surfaces through a series of profiles
    • Chapter 42, “Shadow Curves” describe how to split faces of a collection of bodies into visible and invisible parts
    • Chapter 43, “Creating Emboss Features” describes how to modify a body by creating emboss features using a specified profile.

Contents

All Parasolid functions return error codes that broadly indicate whether the operation has succeeded or failed. This volume provides information on handling error information generated by Parasolid, explains what error codes and failure status codes generated by Parasolid are and how to register error handling functions with Parasolid. It also explains how to register signal handlers to allow the operating system to recover from run-time errors and user interrupts.

This volume contains the following chapters:

    • Chapter 121, “Error Handling” describes handling error and failure status codes
    • Chapter 122, “Signal Handling” describes how to deal with signal handlers and run-time errors that are caught by the operating system.

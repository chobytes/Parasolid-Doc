Contents

Parasolid offers core functionality that supports interoperability between Parasolid-based applications and with non-Parasolid based applications. This volume provides information on these interoperability tools. It explains how models created by other modelers can be imported into Parasolid and how 3D data can be output in a standard format for use in other applications.

This volume contains the following chapters:

    • Chapter 91, “Importing Data” describes how models created by other modelers can be imported into Parasolid either via trimmed surface or B-rep routes
    • Chapter 92, “Exporting Data” describes how model data is exported from Parasolid so it can be used by other non-Parasolid based applications.

Contents

This volume provides information on the functionality that can be used when modelling with wires and sheet bodies. It explains how wire and sheet bodies can be created and manipulated, how free-form sheets can be arbitrarily trimmed to meet designer concepts and then sown together to create a new body. It also provides information on how to work with neutral sheets as well as sheet and surface extension.

This volume contains the following chapters:

    • Chapter 45, “Wire Modeling” describes functionality aimed specifically at wire modeling
    • Chapter 46, “Sheet Modeling” describes functionality aimed specifically at sheet modeling. This is particularly useful when importing surface data into Parasolid.
    • Chapter 47, “Sewing and Knitting” describes how sheets can be sown together
    • Chapter 48, “Mid-Surface Generation” describes how to create and work with neutral sheets
    • Chapter 49, “Extending Sheets And Surfaces” describes how sheets and surfaces can be extended from a set of laminar edges.

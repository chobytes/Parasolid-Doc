Contents

Parasolid supports a wide range of graphical output capabilities that allow users to accurately render large models while minimising memory usage. Parasolid’s extensive model faceting capabilities support visualisation, CAM, CAE, surface analysis and simplification applications. This volume describes how model data is rendered and optionally faceted to generate an image of the model to an output device for your viewing. It also explains the mechanisms for the selective and incremental rendering of models.

This volume contains the following chapters:

    • Chapter 104, “Introduction To Rendering And Faceting” provides a brief introduction to the processes involved in rendering and faceting
    • Chapter 105, “Rendering Functions” describes the rendering functions used in Parasolid
    • Chapter 106, “Rendering Option Settings” describes how to use rendering options
    • Chapter 108, “Facet Mesh Generation” describes how to control the creation of facet meshes
    • Chapter 109, “Faceting Output Via GO” describes the output of facet geometry via the Graphical Output interface
    • Chapter 110, “Tabular Output Of Faceting”describes how to return facet data in tabular format
    • Chapter 111, “Parasolid View Matrices” describe the general form of a view matrix
    • Chapter 107, “Display Tolerances” describe the tolerances used in rendering functions
    • Chapter 112, “Picking Topology” describes how to implement entity selection in an application.

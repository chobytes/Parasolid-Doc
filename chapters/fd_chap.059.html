Contents

Parasolid offers a range of methods that enable CAD users to create thin-walled parts using simple inputs.This volume provides information on these methods: offsetting, hollowing and thickening operations. These powerful modeling operations are used to create new bodies by offsetting new faces and optionally creating a thickening version (when supplying a sheet) or a hollow version (when supplying a solid) of the original body.

This volume contains the following chapters:

    • Chapter 59, “Offsetting” describes how to create new faces from existing ones
    • Chapter 60, “Hollowing” describes how to create a hollowed version of a body
    • Chapter 61, “Thickening” describes how to create a solid body of specified thickness.

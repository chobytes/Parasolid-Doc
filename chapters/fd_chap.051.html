Contents

Parasolid offers unparalleled boolean capabilities that enable you to compare or combine sets of modelling entities in a range of ways. This volume describes Parasolid’s boolean functionality as well as related functionality that either uses boolean functionality or is similar to it. It explains how entities can be imprinted onto sheet bodies and describes the boolean operations that create manifold and general bodies. It explains how instancing can be used to replicate features and how patterning can be used to repeat a pattern of existing features across multiple faces of a model. It also provides information on Parasolid’s sectioning and intersection functionality.

This volume contains the following chapters:

    • Chapter 51, “Imprinting and Curve Projection” describes how curves, faces and planes can be imprinted onto sheet bodies
    • Chapter 52, “Introduction to Booleans” provides an introduction to boolean functionality
    • Chapter 53, “Manifold Booleans” describes boolean operations that result in manifold bodies
    • Chapter 54, “General Booleans” describes boolean operations that result in the creation of general bodies
    • Chapter 55, “Instancing And Patterning” describes functionality that provides faster methods of creating patterns on models. Patterning may be used in situations where instancing is not the best solution.
    • Chapter 56, “Sectioning” describes Parasolid’s support for dividing a body into different sections using surfaces, faces or sheets
    • Chapter 57, “Intersections” describes the support for intersection functions.

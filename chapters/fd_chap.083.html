Contents

This volume provides information on the Convergent Modeling functionality in Parasolid and contains the following chapters:

  • Chapter 83, “Overview Of Convergent Modeling” provides an overview of the facet-enabled API.
  • Chapter 84, “Facet Model Structure” provides information on entities that are unique to Convergent Modeling.
  • Chapter 85, “Converting Between Facet and Classic Geometry” provides information on creating facet bodies.
  • Chapter 86, “Creating PSM Data From Foreign Facet Data” provides information on creating meshes using imported foreign facet data.
  • Chapter 87, “Checking and Repairing Mesh Data”, provides information on how you can check that mesh data is valid and repair any defects found.
  • Chapter 88, “Filling Holes In Facet Bodies” provides information on filling holes in meshes
  • Chapter 89, “Performing Evaluations On Meshes”, provides information on parameterising meshes and mesh normals.
